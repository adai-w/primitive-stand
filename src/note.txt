# 调整对比度饱和度

1. 人脸检测位置矩形
2. 矩形框内画圆或椭圆
3. 从r*4/5处开始模糊，透明度从1到0

做法：
获取人脸框内的所有像素值矩阵M，r为矩形短边的一半，计算每个点M[x,y]离矩形中心点O的距离d，每个点的透明度a={
    0, d > r;
    1, d < 0.8*r;
    5*(r-d)/r, 0.8*r < d < r;
}
